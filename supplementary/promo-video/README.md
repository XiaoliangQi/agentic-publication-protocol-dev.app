# APP Promo Video

A ~70-second promo video for the paper *Agentic Publication Protocol: An
Attempt to Modernize Scientific Publication* (Lu & Qi), built with
[Remotion](https://remotion.dev).

The video opens in a "frozen paper" world (cream, serif, rubber stamps),
flips into a dark "agent" world (terminal, repo tree, workflow, eval
results, research network), and bookends back to paper — now with a
blinking cursor.

## Scenes

0. **Hook card** (2 s) — fully composed at frame 0 so chat apps and players
   pick an informative share thumbnail; `out/thumbnail.png` is the same frame
   at full resolution for platforms that accept custom thumbnails
1. **Cold open** — papers tell you *what*, rarely *how*
2. **The problem** — manuscript stamped STATIC / CAN'T BE UPDATED / HARD TO
   REPRODUCE / KNOW-HOW LOST; 26% reproducibility stat (Stodden et al. 2018)
3. **Flip** — "what if you could ask the paper itself?"
4. **Title** — Agentic Publication Protocol, authors
5. **Repo** — "the repository *is* the publication" + file tree with AGENTS.md
6. **Workflow** — the five `/publish-paper` skills
7. **Results** — 9.25 vs 8.50, 11/11 head-to-head wins
8. **Network** — sparse archive → living research network
9. **Outro** — CTA with the GitHub URL

## Commands

```bash
npm install
npm run dev                                  # Remotion Studio (live preview)
npx remotion render AppPromo out/app-promo.mp4
```

Edit scene copy/timings in `src/scenes/`; scene order and durations are
assembled in `src/Main.tsx`. Colors and fonts (Fraunces + IBM Plex Mono via
Google Fonts) live in `src/theme.ts`.

## Music

The score (`public/music.wav`) is an original synthesized track generated by
`music/generate-music.mjs` — fully license-free and timed to the scenes:
heartbeat + booms on the rubber stamps, a riser through the terminal flip,
a four-on-the-floor drop when the title lands, and a calm reprise for the
outro. Regenerate after changing scene timings:

```bash
node music/generate-music.mjs
```

(Scene boundary times are hard-coded near the top of that script.)
To swap in a real track instead, replace the `<Audio>` tag in `src/Main.tsx`.
